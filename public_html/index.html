About PyWolf

PyWolf is a Python WebApp that queries Wolfram|Alpha for your input and automatically gives your result displayed with Step-by-Step solution. It is open source and only requires yo to apply to an AppID at Wolfram|Alpha (for free) if you are interested in porting it.

I do not own Wolfram|Alpha and I'm not responsible for the output displayed by your queries.

The template used to make this Website was a courtesy of Start Bootstrap, to whom I'm very grateful.

This project is and will always be free.
